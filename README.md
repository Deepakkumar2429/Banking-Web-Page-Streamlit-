# 🏦 Bank Management System (Streamlit)

This project is a Bank Management Web Application built using Python, Streamlit, and SQLite. It simulates basic banking operations through an interactive web interface. The application allows users to create accounts, log in securely, and perform banking activities such as depositing money, withdrawing money, checking account balance, and viewing a mini statement.

The system uses SQLite as the backend database to store user account details including username, password, PIN, and account balance. Streamlit session state is implemented to manage user login sessions and provide a dynamic user experience.

This project demonstrates the implementation of Python programming, database integration, authentication handling, and web application development using Streamlit.

# 🚀 Features

- User Signup (Account Creation)

- Secure Login Authentication

- Deposit Money

- Withdraw Money

- Check Account Balance

- Mini Statement

- SQLite Database Integration

- Session Management using Streamlit

# 🛠 Technologies Used

- Python

- Streamlit

- SQLite3
